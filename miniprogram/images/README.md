# 图片资源说明

本目录用于存放小程序所需的图片资源。

## 需要准备的图片文件

### TabBar图标（必须）
- `tab_home.png` - 首页图标（未选中）
- `tab_home_active.png` - 首页图标（选中）
- `tab_map.png` - 导览图标（未选中）
- `tab_map_active.png` - 导览图标（选中）
- `tab_camera.png` - 识别图标（未选中）
- `tab_camera_active.png` - 识别图标（选中）
- `tab_profile.png` - 我的图标（未选中）
- `tab_profile_active.png` - 我的图标（选中）

**规格**：建议 81x81 像素，PNG格式

### 其他图片
- `ai_avatar.png` - AI助手头像（圆形，128x128像素）
- `default_avatar.png` - 默认用户头像
- `marker.png` - 地图标记图标（红色图钉，50x50像素）
- `share_bg.png` - 分享封面图（建议 500x400 像素）

## 临时解决方案

如果暂时没有图片文件，可以：
1. 使用 emoji 表情符号代替（如 🍵🏠🗺️📷👤）
2. 或在开发工具中跳过图片加载测试

## 图标推荐来源

- TabBar图标：https://www.iconfont.cn/
- 地图标记：可使用系统内置marker或自定义红色图钉
